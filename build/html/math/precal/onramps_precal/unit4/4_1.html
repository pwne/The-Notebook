4.1 Working with Identities¶

Definition:

An Identity is an equation that is always true regardless of the numbers substituted for the variables.

4.1.1 Identifying Identities¶

Identities can be Identifying by looking for consistent patterns.

Some common Identities are:

Distributive Identity:

\(a(b+c) = ab + bc\)

Commutative Identity:

\(a + b = b + a \\ a \cdot b = b \cdot a\)

\(\frac{a}{a} = 1\) (unless \(a = 0\), then undefined)

Note

To prove that an equations NOT an Identity, find one (1) case where it is NOT true.

\(a^2 + b^2 = c^2\) is an identity ONLY IF \(a\) and \(b\) are the legs of a right triangle and \(c\) is the hypotenuse

\(x=\frac{\sqrt{b^2-4(ac)}}{2a}\) is an identity ONLY IF \(ax^2+bx+c=0\).

4.1.2 More Trigonometric Properties¶

Reciprocal Properties¶

The Reciprocal Properties involve taking the reciprocal of the three standard Trigonometric Functions (\(\sin, \cos, \tan\)) to create their inverses (\(\csc, \sec, \cot\)). Below are the Reciprocal Properties:

  1. \(\csc{\theta} = \frac{1}{\sin{\theta}}; \hspace{10 pt} \sin{\theta} = \frac{1}{\csc{\theta}}\)

  2. \(\sec{\theta} = \frac{1}{\cos{\theta}}; \hspace{10 pt} \cos{\theta} = \frac{1}{\sec{\theta}}\)

  3. \(\cot{\theta} = \frac{1}{\tan{\theta}}; \hspace{10 pt} \tan{\theta} = \frac{1}{\cot{\theta}}\)

Quotient Properties¶

The Quotient Properties apply to only the \(\tan\theta \text { and } \cot\theta\) functions:

  1. \(\tan\theta = \displaystyle \frac{\sin\theta}{\cos\theta}\)

  2. \(\cot\theta = \displaystyle \frac{\cos\theta}{\sin\theta}\)

Note

The \(\cot\) function is the reciprocal of the \(\tan\) function.

Pythagorean Properties:¶

Using the \(\cos^2{\theta}+\sin^2{\theta} = 1\) identity based on the Pythagorean Theorem for a reference triangle inside a Unit Circle we can derive more Identities from it:

  • \(\sin^2\theta + \cos^2\theta = 1 \hspace{20 pt}\)

  • \(\sin^2\theta = 1 - \cos^2\theta\)

  • \(\cos^2\theta = 1 - \sin^2\theta\)

  • \(\sec^2\theta - \tan^2\theta = 1 \hspace{20 pt}\)

  • \(\sec^2\theta = \tan^2\theta + 1\)

  • \(\tan^2\theta = \sec^2\theta - 1\)

  • \(\csc^2\theta - \cot^2\theta = 1 \hspace{20 pt}\)

  • \(\csc^2\theta = \cot^2\theta + 1\)

  • \(\cot^2\theta = \csc^2\theta - 1\)
