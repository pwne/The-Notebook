3.0 Review Exponents and Logarithms¶

3.0.1 Exponents¶

Exponents are a type of mathematical operation, written as \(b^n\), that tells us how many times (\(n\)) to multiply a number (\(b\)) by itself.

Exponent Properties¶

Below are some Exponent Properties that will be helpful when dealing with exponents.

Rule

Example

\(a^m \cdot a^n = a^{m + n}\)

\(x^2 \cdot x^5 = x^7\)

\((ab)^m = a^m \cdot b^m\)

\((xy)^3 = x^3y^3\)

\((a^m)^n = a^{mn}\)

\((x^2)^3 = x^6\)
If \(m > n\), then
\(\frac{a^m}{a^n} = a^{m-n}\)

\(\frac{x^5}{x^3} = x^2\)
If \(n > m\), then
\(\frac{a^m}{a^n} =\) \(\frac{1}{a^{n-m}}\)

\(\frac{x^3}{x^5} =\) \(\frac{1}{x^2}\)

\(\lgroup\frac{a}{b}\rgroup^m =\) \(\frac{a^m}{b^m}\)

\((\frac{x}{y})^3 =\) \(\frac{x^3}{y^3}\)

\(a^{-m} = \frac{1}{a^m}\)

\(3^{-2} = \frac{1}{9}\)

\(\frac{1}{a^{-m}} = a^m\)

\(\frac{1}{2^{-3}} = 8\)
\(a^0 = 1\), as long as \(a \neq 0\).
\(0^0\) is undefined.

\(5^0 = 1\)

\(a^{\frac{m}{n}} = \sqrt[n]{a^m}\)

\(9^{\frac{3}{2}} = 27\)

Simplifying Expressions¶

Simplifying Expressions is the process of condensing long and hard to read Expressions into short and easy to read ones. Use the Exponent Properties to help you simplify expressions.

Examples:

\(x^3 \cdot x^4 = x^7\)

\(\frac{x^6}{x^2} = x^4\)

\((a^3b^5)/(ac^3) = \frac{a^3b^5}{ac^3} = \frac{a^2b^5}{c^3}\)

\((2^2)^3 = 2^6 = 2 \cdot 2 \cdot 2 \cdot 2 \cdot 2 \cdot 2 = 64\)

\((4x^2y^3)^3 = 4^3x^6y^9 = 64x^6y^9\)

3.0.2 Logarithms¶

Logarithms are the inverse functions of Exponents, commonly written as \(\log_b{b^x} = x\), and tell us what exponent is used to get a Number given the base

Logarithmic Forms¶

There are 2 ways to write out Logarithms:

Exponential Form¶

Syntax:

\(b^x = n\)

Where \(b\) is the Base, \(x\) is the Exponent, and \(n\) is the Answer.

Example:

\(5^4 = 625\)

Logarithmic Form¶

Syntax:

\(\log_b{b^x} = x\)

Where \(b\) is the Base, \(x\) is the Exponent, and \(b^x\) is the Answer.

Example:

\(log_5{625} = 4\)

Logarithmic Properties¶

Logarithms:

\(y = \log_b x\) if and only if \(b^y = x\)

Below are some Logarithmic Properties that will be helpful when dealing with Logarithms.

  1. \(\log_b{M} + \log_b{N} = \log_b{(MN)}\)

  2. \(\log_b{(M)} - \log_b{(N)} = \log_b{\frac{M}{N}}\)

  3. \(\log_b{(M^k)} = k\log_b{M}\)

  4. \(\log_a{x} = \frac{\log_b{x}}{\log_x{a}}\) (Change of Base Formula)

Note

The Properties above only apply if the following rules are met:

\(b > 0\), \(b \neq 1\), \(M > 0\), \(N > 0\), and Natural Number \(k\)

Condensing and Expanding Logarithms¶

Use the Logarithmic Properties when Condensing and Expanding Logarithms:

Condensing Logarithms¶

Examples:

\(\log_4{3} + \log_4{8} = \log_4{24}\)

\(\log_2{225} - \log_2{5} + \log_2{3} = \log_2{\frac{225}{5} \cdot 3} = \log_2{135}\)

\(4\log_2{3} = \log_2{81}\)

\(\frac{1}{2}\ln{x} + 2\ln{y} - 4\ln{z} = \ln{\frac{\sqrt{x} \cdot y^2}{z^4}}\)

\(2 - \log_4{x} = \log_4{16} - \log_4{x} = \log_4{\frac{16}{x}}\)

Expanding Logarithms¶

Examples:

\(\log_3{x^2y^3} = \log_3{x^2} + \log_3{y^3}\)

\(\log_2{\frac{8x}{z}} = \log_2{8x} - \log_2{z}\)

\(\log_5{\frac{5y^3}{x^2}} = 1 + 3\log_5{y} - 2\log_5{x}\)
